🚛 Azkar Mudanzas — Guía de Instalación PWA

Tu app de gestión de presupuestos en Windows y Android

0 ¿Qué incluye este paquete?

index.html — La aplicación completa
manifest.json — Configuración PWA
sw.js — Service Worker (funciona offline)
icons/ — Iconos para todas las plataformas
INSTALAR.html — Esta guía

1 Subir a un servidor web (obligatorio para PWA)

La PWA necesita servirse desde HTTPS. Tienes varias opciones GRATIS:
Opción A: GitHub Pages (recomendada, gratis)
1. Crea una cuenta en github.com (si no tienes) 2. Crea un repositorio nuevo llamado "azkar-app" 3. Sube todos los archivos de esta carpeta 4. Ve a Settings → Pages → Source: "main" → Save 5. Tu app estará en: https://tuusuario.github.io/azkar-app/
Opción B: Netlify (gratis, más fácil)
1. Ve a netlify.com y crea cuenta 2. Arrastra la carpeta "azkar-pwa" completa al panel 3. ¡Listo! Te da una URL tipo: https://azkar-xxx.netlify.app
Opción C: Firebase Hosting (gratis)
npm install -g firebase-tools firebase login firebase init hosting firebase deploy
Opción D: Servidor local para probar
python -m http.server 8080 # Abre http://localhost:8080 en Chrome

2 Instalar en Windows WINDOWS

Una vez la app está en un servidor HTTPS:
1. Abre Chrome/Edge y ve a tu URL 2. Verás el banner "Instalar Azkar Mudanzas" abajo 3. Pulsa "INSTALAR" 4. ¡Listo! Aparecerá en tu escritorio y menú inicio Alternativa: Click en el icono ⊕ en la barra de Chrome O en Edge: ··· → Aplicaciones → Instalar este sitio
Tip: En Edge también puedes ir a edge://apps para gestionar tus PWAs instaladas.

3 Instalar en Android ANDROID

1. Abre Chrome en tu Android y ve a tu URL 2. Verás el banner "Instalar Azkar Mudanzas" 3. Pulsa "INSTALAR" (o "Añadir a pantalla de inicio") 4. ¡Listo! Aparecerá como app en tu teléfono Alternativa: Chrome → ⋮ → "Instalar aplicación" O: Chrome → ⋮ → "Añadir a pantalla de inicio"
Tip: En Samsung Internet también funciona: ☰ → Añadir página a → Pantalla de inicio.

4 Configurar sincronización Firebase (opcional)

Para que los presupuestos se sincronicen entre tu PC y tu móvil:
1. Ve a https://console.firebase.google.com 2. Crea un proyecto nuevo (nombre: "azkar-app") 3. Ve a Firestore Database → Crear base de datos → Modo test 4. Ve a Configuración del proyecto → Tus apps → Web () 5. Registra la app y copia los datos de configuración 6. En la app Azkar: 📋 MIS PRESUPUESTOS → ⚙️ Configurar Firebase 7. Pega los datos y pulsa "Conectar"
Importante: Usa el mismo proyecto Firebase en todos tus dispositivos. Los datos se sincronizan en tiempo real automáticamente.

5 Funcionalidades de la App

📋 Mis Presupuestos: Gestiona todos tus presupuestos guardados. Botón en la barra superior.

💾 Guardado automático: Se guarda automáticamente cada 60 segundos mientras trabajas.

⚡ Modo offline: Funciona sin conexión. Los cambios se sincronizan al volver a estar online.

☁️ Sincronización: Con Firebase activado, los presupuestos aparecen en todos tus dispositivos al instante.

🖨️ Exportar PDF: Usa el botón "IMPRIMIR PDF" para generar documentos para el cliente.
